Font Awesome Animation

Simple animations using FontAwesome and some CSS3.

 Getting started

Does not work with < IE10...

Download

Manually

The files can be downloaded from:

  • Minified CSS for production usage
  • Un-minified CSS for development

With Bower

$ bower install font-awesome-animation

Installation

Include the JS and CSS file in your index.html file: 

<link rel="stylesheet" href="font-awesome-animation.min.css">

Installation

Include the JS and CSS file in your index.html file: 

<link rel="stylesheet" href="font-awesome-animation.min.css">

 Usage

Add the desired CSS class to the icon (or any element of your DOM).

For parent hover, add the CSS class faa-parent and animated-hover on the parent.

You can regulate the speed of the animation by adding the CSS class faa-fast or faa-slow.
